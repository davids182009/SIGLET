GeoAnalytics distribuisce il lavoro di analisi in tutto il server ArcGIS GeoAnalytics. Ciò consente di analizzare un maggior numero di dati più velocemente sfruttando la potenza di più computer. Al momento sono disponibili i seguenti strumenti:

  • Aggrega Punti
  • Join delle feature
  • Ricostruisci tracce
  • Riepiloga attributi
  • Riepiloga entro
  • Rileva Eventi
  • Trova posizioni simili
  • Calcolo densità
  • Trova hot spot
  • Crea buffer
  • Copia in Data Store
  • Calcola campo
  • Geocodifica posizioni da tabella

GeoAnalytics si concentra sull'analisi di dati di grandi dimensioni, con una particolare enfasi su i modelli sia spaziali che temporali.

Riepiloga dati

Il set di strumenti Riepiloga dati contiene strumenti che consentono di calcolare totali, lunghezze, aree e statistiche descrittive di base di feature e dei relativi attributi in aree o altre feature vicine.

  • Aggrega punti consente di calcolare le statistiche su punti inclusi in aree o contenitori specifici.
  • Join delle feature consente di calcolare le statistiche su feature che condividono una relazione spaziale, temporale o di attributi con altre feature.
  • Ricostruisci tracce consente di calcolare statistiche su punti o poligoni che appartengono alla stessa traccia e di ricostruire gli input in tracce.
  • Riepiloga attributi consente di calcolare le statistiche su feature o dati tabulari che condividono attributi.
  • Riepiloga entro consente di calcolare le statistiche per le feature di area e per gli attributi sovrapposti tra loro.

Aggrega punti

Utilizzando un layer di feature puntuale e un layer di feature areali o dei contenitori definiti da una distanza specifica, questo strumento determina i punti o i contenitori inclusi in ciascuna area e calcola le statistiche su tutti i punti all'interno di ogni area o contenitore. Questo strumento consente anche di applicare i periodi di tempo.

Ad esempio:

  • Date le posizioni dei punti relativi a reati, è possibile calcolare il numero di reati per regione o per un altro tipo di distretto amministrativo.
  • È possibile calcolare i ricavi minimi e massimi di negozi in franchising utilizzando contenitori di 100 km.


Join delle feature

Utilizzando Feature Layer o dati tabulari, è possibile unire le feature e i record in base a relazioni specifiche tra layer di input o tabelle. I join saranno determinati da relazioni spaziali, temporali e di attributi ed è possibile, facoltativamente, calcolare statistiche di riepilogo.

Ad esempio:

  • Date le posizioni puntuali dei reati in un determinato tempo, unire i dati dei reati tra di loro, specificando una relazione spaziale dei reati, entro 1 km l'uno dall'altro e nell'intervallo di 1 ora l'uno dall'altro, per determinare se esiste una sequenza di reati vicini tra loro nello spazio e nel tempo.
  • Data una tabella di codici postali con informazioni demografiche e feature areali che rappresentano edifici residenziali, unire le informazioni demografiche agli edifici residenziali, in modo che ciascuno di essi contenga le informazioni.


Ricostruisci tracce

Utilizzando un layer di punti abilitati per variazioni temporali o feature poligonali che rappresentano un istante nel tempo, questo strumento determina quali feature di input appartengono a una traccia e ordinerà gli input in modo sequenziale nel tempo. Le statistiche vengono calcolate facoltativamente per le feature di input all'interno di ogni traccia.

Ad esempio:

  • Date le posizioni puntuali e l'ora delle misurazioni degli uragani, calcolare la velocità media del vento e la pressione massima del vento dell'uragano.
  • Usa misurazioni GPS di navi per ricostruire percorsi navali come linee.


Riepiloga attributi

Utilizzando feature o dati tabulari, questo strumento riepiloga le statistiche per campi specifici.

Ad esempio:

  • Date le posizioni dei negozi di alimentari con un campo COMPANY_NAME, riepilogare i negozi in base al nome della società per determinare le statistiche per ciascuna società.
  • Data una tabella di negozi di alimentari con i campi COMPANY_NAME e COUNTY, riepilogare i negozi in base al nome della società e della regione per determinare le statistiche per ciascuna società in ogni regione.


Riepiloga entro

Consente di trovare aree (e parti di aree) che si sovrappongono tra due layer e di calcolare le statistiche sulla sovrapposizione.

Ad esempio:

  • Dati un layer di aree spartiacque e un layer di aree di utilizzo del suolo per tipo di utilizzo, è possibile calcolare la superficie totale in acri del tipo di utilizzo del suolo per ogni spartiacque.
  • Dati un layer di particelle in una regione e un layer di confini di città, è possibile riepilogare il valore medio delle particelle non occupate in ogni città.


Trova posizioni

Questi strumenti consentono di identificare le aree conformi a un certo numero di criteri specificati.

  • Rileva Eventi funziona con layer abilitati per variazioni temporali le cui feature soddisfano una condizione specificata.
  • Geocodifica Posizioni da Tabella consente di convertire indirizzi da una tabella in coordinate.
  • Trova Posizioni Simili consente di trovare le posizioni più simili a una o più posizioni di riferimento in base ai criteri specificati.

Rileva Eventi

Questo strumento funziona con un layer abilitato per variazioni temporali di punti, linee, aree o tabelle che rappresentano un istante nel tempo. Con l'uso di feature ordinate in sequenza, denominate tracce, questo strumento determina le feature che sono eventi di interesse. Gli eventi vengono determinati dalle condizioni specificate.

Ad esempio:

  • Rilevare le misurazioni degli uragani di Categoria 4 in base all’attributo di velocità del vento.
  • Utilizzando le misurazioni GPS del movimento della nave con un campo che rappresenta la velocità della nave, trovare le misurazioni superiori di 1,5 volte alla media delle precedenti cinque misurazioni.


Geocodifica Posizione da Tabella

Consente di convertire indirizzi in coordinate. Utilizzare questo strumento su tabelle di condivisione file Big Data.


Trova località simili

In base ai criteri specificati, trovare posizioni simili calcolando la somiglianza dei luoghi presenti nel layer di ricerca con uno o più posizioni di riferimento.

Ad esempio:

  • Trovare i dieci negozi più simili esaminando il numero dei dipendenti e le vendite annuali.
  • Trovare le cento città più simili esaminando la relazione tra popolazione, crescita annuale ed entrate fiscali.


Analizza modelli

Questi strumenti consentono di identificare, quantificare e visualizzare i modelli spaziali nei dati.

  • Calcolo densità consente di valutare quantità note di alcuni fenomeni e di diffondere tali quantità nella mappa.
  • Trova hot spot consente di identificare cluster di rilevanza statistica nel modello spaziale dei dati.

Calcolo Densità

Lo strumento Calcolo densità consente di creare una mappa di densità a partire da feature puntuali diffondendo nella mappa quantità note di alcuni fenomeni (rappresentati come attributi di punti). Il risultato è un layer di aree che rappresentano la densità.

Ad esempio:

  • Per calcolare le densità relative agli ospedali presenti in una determinata area. Nel layer dei risultati verranno indicate le aree con maggiore o minore accessibilità agli ospedali e tali informazioni saranno utili per stabilire dove costruire nuove strutture.
  • Per identificare aree ad alto rischio di incendio sulla base delle posizioni in cui nel tempo si sono verificati incendi di aree boschive.
  • Per individuare le comunità più lontane dalle principali autostrade e pianificare le aree in cui costruire nuove strade.


Trova hot spot

Lo strumento Trova Hot Spot consente di determinare la presenza di cluster statisticamente rilevanti nel modello spaziale dei dati.

  • I punti (reati, alberi, incidenti stradali) sono realmente organizzati in cluster? Come si può verificarlo?
  • È stato effettivamente individuato un hot spot di rilevanza statistica (ai fini della spesa, della mortalità infantile, di punteggi di test elevati) oppure la mappa raffigurerebbe una situazione diversa se venisse cambiata la simbologia?
Lo strumento Trova hot spot consente di rispondere con sicurezza a queste domande.


Usa prossimità

Questi strumenti consentono di rispondere a una delle domande più comuni dell'analisi spaziale: "Che cosa è vicino a cosa?"

  • Crea buffer consente di creare aree a una distanza specificata dalle feature.

Crea Buffer

Un buffer è un'area che copre una data distanza da una feature puntuale, lineare o poligonale.

Ad esempio:

  • Utilizzando feature lineari per i fiumi, eseguire il buffer di ciascun fiume per 50 volte la larghezza del fiume al fine di determinare un confine rivierasco proposto.
  • Date le aree che rappresentano i paesi, eseguire il buffer di ciascun paese per 200 miglia nautiche al fine di determinare il confine marittimo.


Gestisci dati

Questi strumenti vengono utilizzati per la gestione quotidiana dei dati geografici e tabulari.

  • Calcola campo crea un nuovo campo o modifica il campo esistente e crea un nuovo layer nei contenuti.
  • Copia in Data Store copia i dati in ArcGIS Data Store e crea un layer nei contenuti.

Copia in Data Store

Consente di copiare un Feature Layer di input o una tabella in un ArcGIS Data Store e creare un layer in ArcGIS Enterprise.

Ad esempio:

  • Copiare una raccolta di file .csv in una condivisione file Big Data nello Spatiotemporal Data Store per la visualizzazione.
  • Copiare le feature nell'estensione della mappa corrente che sono memorizzate nello Spatiotemporal Data Store nel Data Store relazionale.


Calcola Campo

Calcola valori per campo nuovo o esistente e crea un nuovo layer in ArcGIS Enterprise.

Ad esempio:

  • Modificare un campo esistente denominato totale in modo che sia la somma delle entrate dai campi totale_2016, totale_2017, e totale_2018.
  • Creare un nuovo campo per classificare i livelli di pericolo in base a valori di campo quali velocità del vento e inquinante.
